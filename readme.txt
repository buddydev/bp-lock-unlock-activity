BuddyPress Lock Unlock Activity allows users to close/open activity for commenting.

== Description ==

BuddyPress Lock Unlock Activity plugin allows users on BuddyPress based site to lock their activity and control
if others can comment on his/her activity.

It may save flame wars on your social network and make your users feel more in control of their activity.

Features:-

*   Allows user to close activity. By closing activity, commenting is disabled on the activity.
*   Allows users to unlock(open) activity. Other users can comment on unlocked activity.
*   Translatable
*   Simple and easy to use.

Fo more details, Please see the [plugin page on BuddyDev](https://buddydev.com/plugins/bp-lock-unlock-activity/)
== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/bp-lock-unlock-activity` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the Lock Unlock Activity for BuddyPress through the 'Plugins' screen in WordPress
1. That's all. Visit the activity to see it in action.
